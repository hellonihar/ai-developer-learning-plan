# AI Developer Learning Plan

A structured, hands-on curriculum to build production-ready AI engineering skills — from Python foundations to deploying LLM-powered agents.

## Phases

| Phase | Topic | Projects |
|-------|-------|----------|
| 1 | Foundations | Python scripts, Git collaboration, Flask + Docker |
| 2 | LLM Prompting | FAQ chatbot, Resume enhancer, Doc QA (LangChain RAG) |
| 3 | ML Basics | Linear regression, K-means clustering, Spam classifier |
| 4 | RAG & Agents | Semantic search (FAISS/Pinecone), Enterprise RAG, Workflow agent |
| 5 | Deployment | FastAPI service, Monitoring dashboard, Secure pipeline |

## Getting Started

```bash
git clone https://github.com/<your-org>/ai-developer-learning-plan.git
cd ai-developer-learning-plan
pip install -r requirements.txt
```

Each phase folder contains its own sub-projects with individual READMEs and starter code.

## Goals

- Build real AI applications, not just notebooks
- Learn prompt engineering, RAG, and agent orchestration
- Deploy with monitoring and security best practices
- Create portfolio-ready demo artifacts
